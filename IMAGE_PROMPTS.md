Recommended dimensions: **1200x630** for social sharing, **1920x1080** for hero images.

## Tips for Best Results

1. **Always specify "no text"** — you'll add text overlays yourself for better control.
2. **Request dark background** — matches the landing page theme.
3. **Use the exact hex colors** — `#0a0a0f` (bg), `#6366f1` (indigo), `#06b6d4` (cyan), `#22c55e` (green).
4. **Try multiple generations** — AI image tools vary in quality; pick the best result.
5. **Post-process** — Adjust brightness/contrast to match the page's dark aesthetic.
6. **Format** — Save as `.webp` or compressed `.png` for web performance.
